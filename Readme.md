## Discord Music Bot
## This code ate every Discord music bot for an evening snack 

### Setting Up

1. Clone the repo:
   ```bash
   git clone https://github.com/SwizDev01/TuneWave.git
   cd kronix-music-bot
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the bot:
   ```bash
   node src/index.js
   ```
